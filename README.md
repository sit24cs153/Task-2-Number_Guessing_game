# Task-2-Number_Guessing_game
🎯 Guess Number Game (Java)  This console-based Java game challenges users to guess a random number (1–100) within 5 attempts across 3 rounds 🎮. It provides hints like “Too High” or “Too Low” 📢 and rewards points based on remaining attempts 🏆. Built using Random, Scanner, loops, and conditionals.
