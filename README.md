🎯 Task 2 Guess Number Game (Java Project)

This repository contains a console-based **Guess Number Game** developed using Java. The game challenges users to guess a randomly generated number within limited attempts, making it fun and interactive while demonstrating core programming concepts.


# 🎯 Objective

The objective of this project is to build a simple yet engaging Java application that demonstrates:

* 🎲 **Random Number Generation:** Using Java’s Random class
* 🔁 **Control Flow:** Implementing loops and conditional statements
* 🧠 **Logical Thinking:** Providing hints like "Too High" or "Too Low"
* 🏆 **Score Calculation:** Rewarding users based on performance


# 🛠️ Features & Functionality

* 🎮 **Multiple Rounds:** The game runs for 3 rounds
* 🔢 **Random Number (1–100):** Generated for each round
* 🎯 **Limited Attempts:** 5 guesses per round
* 📢 **Hints System:** Guides user with "Too High" or "Too Low"
* ✅ **Win Detection:** Ends round when guessed correctly
* ❌ **Lose Condition:** Reveals number after failed attempts
* 🏆 **Scoring System:** Points awarded based on remaining attempts
* 🚪 **Game Summary:** Displays total score at the end

# 💻 Tech Stack

* **Language:** Java ☕
* **Concepts Used:**

  * Random class
  * Scanner (user input)
  * Loops (for & nested loops)
  * Conditional statements (if-else)

# 🚀 How to Run

1. Compile the program

   bash
   javac GuessNumberGame.java
   
2. Run the application

   bash
   java GuessNumberGame
   


# 🎮 How to Play

* Enter a number between **1 and 100** 🔢
* You have **5 attempts** per round ⏳
* The system will guide you with hints 📢
* Try to guess correctly and score maximum points
